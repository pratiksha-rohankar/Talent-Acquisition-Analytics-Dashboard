# Data Files

This folder contains the dataset used for the Talent Acquisition Analytics Dashboard project.

## Dataset Source

The original dataset was downloaded from Kaggle.

## Data Preparation

The dataset was cleaned and prepared using Microsoft Excel:

* Removed duplicate records
* Checked for missing values
* Standardized data formats
* Converted data into structured tables
* Prepared data for SQL and Power BI analysis

## Files Included

* recruitment_mydata.csv (cleaned dataset)
* Recruitment_Dataset.xlsx (Excel working file)

## Dataset Contents

The dataset includes information such as:

* Candidate Details
* Resume Scores
* Technical Scores
* Personality Scores
* Recruitment Strategies
* Interview Results
* Hiring Decisions
* Candidate Drop-off Stages

This processed dataset was used for SQL analysis and Power BI dashboard development.
