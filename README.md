# Talent Acquisition Analytics Dashboard

## Project Overview
This project analyzes the recruitment process using Excel, SQL, and Power BI. The dashboard helps identify hiring trends, candidate drop-off stages, recruitment strategy effectiveness, and key factors influencing candidate selection.

## Tools Used
- Excel
- SQL
- Power BI

## Key Metrics
- Total Applicants
- Candidates Selected
- Selection Rate
- Average Resume Score

## Dashboard Insights
- Candidate Hiring Journey
- Top Candidate Rejection Reasons
- Hiring Success by Recruitment Strategy
- High Performer Selection Analysis

## Files Included
- Talent Acquisition Analytics Dashboard.pbix
- Recruitment_Dataset.xlsx
- recruitment_mydata.csv
- SQL_Queries.sql
